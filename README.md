# AiM Studio — Production AI Infrastructure

Portfolio and service offering site for **Ankit Panicker / AiM Studio**.
Showcases fixed-price, end-to-end AI platform delivery, production architecture diagrams, and white-labeling services.

## Overview
- **Author**: Ankit Panicker
- **Focus**: AI Systems Engineering, Local AI, CI/CD, Multitenancy, and Infrastructure.
- **Tech Stack**: HTML, CSS, JavaScript (Vanilla, no frameworks), Firebase Hosting.

## Deployments
- **GitHub**: [mrankitpanicker/aimsystems](https://github.com/mrankitpanicker/aimsystems)
- **Live Demo**: [aimsystems.web.app](https://aimsystems.web.app)
- **Live App**: [aimstudio.co.in/app](https://aimstudio.co.in/app)

## Features
- Custom SVG diagrams for HLD (High-Level Design).
- Responsive, premium dark-mode interface with vanilla CSS.
- Zero dependencies for rapid loading.
